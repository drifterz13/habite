## Features

- [x] User profile
    - [x]  Item managements
    - [ ] Item equipment
    - [ ]  Level progression
    - [ ]  Stats enhancement
    - [ ]  Stamina system to fight monster
- [x] Quest / Task
    - [x]  Create quest and task
    - [x]  Delete quest
    - [x]  Start quests
    - [x] Complete quest
    - [x] Grant quest rewards
- [x] Monster / Boss
    - [x]  Damaging boss
    - [x]  Spawning boss
    - [x]  Grant boss rewards
    - [ ]  Monster fight player back
- [ ] Shopping and trading system
    - [ ]  User can buy items
    - [ ]  User can sell items
- [ ] Item system
    - [ ]  Gear tiers
    - [ ]  More variety of items (eg. consumable items)
